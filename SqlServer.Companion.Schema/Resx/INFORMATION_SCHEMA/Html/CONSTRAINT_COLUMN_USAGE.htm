CONSTRAINT_COLUMN_USAGE

Contains one row for each column, in the current database, that has a constraint defined on it. This information schema view returns information about the objects to which the current user has permissions. The INFORMATION_SCHEMA.CONSTRAINT_COLUMN_USAGE view is based on the sysobjects, syscolumns, and systypes system tables.

To retrieve information from these views, specify the fully qualified name of INFORMATION_SCHEMA view_name.
